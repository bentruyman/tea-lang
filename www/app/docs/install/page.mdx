  Tea is built from source with the standard Rust toolchain. You’ll also need **Bun** for codegen, **Make** for repo workflows, and optionally **LLVM** for AOT compilation.

```bash
git clone https://github.com/bentruyman/tea-lang.git
cd tea-lang
make setup
cargo build --release
make install
tea --version
```

If you’d rather skip the install step, you can run Tea directly from the repo during development:

```bash
cargo run -p tea-cli -- examples/echo/main.tea hello tea
```
