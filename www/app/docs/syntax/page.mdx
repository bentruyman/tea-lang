# Syntax Basics

Learn the fundamentals of Tea's syntax. Tea uses a clean, readable syntax with Python-like indentation
awareness and Ruby-inspired block endings.

## Variables

Tea has two kinds of bindings: mutable variables with `var` and immutable constants with `const`.

<CodeCard title="Mutable Variables" description="Use var for values that can be reassigned.">
{`var count = 0
count = count + 1

var name = "Tea"
name = "Tea Language"

var numbers = [1, 2, 3]
numbers = [4, 5, 6]`}
</CodeCard>

<CodeCard title="Constants" description="Use const for values that should never change.">
{`const PI = 3.14159
const APP_NAME = "MyApp"
const MAX_RETRIES = 3

# Constants can also hold functions
const double = |x: Int| => x * 2`}
</CodeCard>

<CodeCard title="Type Annotations" description="Types are inferred automatically, but you can add explicit annotations.">
{`# Type inference
var count = 42          # Inferred as Int
var name = "Tea"        # Inferred as String

# Explicit types
var count: Int = 42
var name: String = "Tea"

# Optional types
var maybe_name: String? = nil`}
</CodeCard>

## Control Flow

<CodeCard title="If Statements" description="Conditionals use if, else, and end.">
{`var score = 85

if score >= 90
  @println("Grade: A")
else
  @println("Grade: B or lower")
end

# Nested conditions
if score >= 90
  @println("A")
else
  if score >= 80
    @println("B")
  else
    @println("C or lower")
  end
end`}
</CodeCard>

<CodeCard title="If Expressions" description="Use if() for ternary-style expressions that return values.">
{`const is_admin = true
const role = if(is_admin) "admin" else "user"

var x = 10
var y = if(x > 5) x * 2 else x

# With function calls
const result = if(true) double(5) else double(10)`}
</CodeCard>

<CodeCard title="While Loops">
{`var count = 0

while count < 5
  @println(count)
  count = count + 1
end`}
</CodeCard>

## Strings

<CodeCard title="String Literals">
{`var simple = "Hello, World!"
var with_escape = "Line 1\\nLine 2"
var with_quotes = "She said \\"Hello\\""`}
</CodeCard>

<CodeCard title="String Interpolation" description="Use backticks for interpolated strings with ${...} expressions.">
{`var name = "Tea"
var greeting = \`Hello, \${name}!\`

var x = 5
var result = \`Next number: \${x + 1}\`

# Function calls in interpolation
def double(n: Int) -> Int
  n * 2
end

@println(\`Doubled: \${double(21)}\`)  # "Doubled: 42"`}
</CodeCard>

<CodeCard title="String Operations">
{`var s = "Hello"

# Length
var len = @len(s)          # 5

# Indexing (0-based)
var first = s[0]           # "H"
var last = s[4]            # "o"

# Slicing
var sub = s[0..3]          # "Hel" (exclusive end)
var sub2 = s[1...4]        # "ell" (inclusive end)

# Concatenation
var full = s + " World"    # "Hello World"`}
</CodeCard>

## Printing Output

</CodeCard>
